## 服务形态

### 1. C 端免费对话（智谱清言）
- 网址：https://chatglm.cn/
- 免费使用 GLM-5 / GLM-5.1
- 支持模式：思考、联网、Agent、研究模式、PPT 模式、数据分析
- 支持功能：AI 画图、AI 阅读、AI 生视频、云知识库、GLM-Claw 智能体

### 2. API 按量计费（开放平台）
- 网址：https://bigmodel.cn/
- 注册赠送 2000 万 tokens
- 支持数十个模型的 API 调用
- 兼容 OpenAI API 格式
- 提供多种 SDK

### 3. Coding Plan 编码订阅套餐
- 专为 AI 编码打造
- 支持 GLM-5.1、GLM-5-Turbo、GLM-4.7、GLM-4.5-Air
- 兼容工具：Claude Code、Kilo Code、OpenClaw、OpenCode、TRAE、CodeBuddy
- 55+ tokens/秒生成速度
- 三档：Lite / Pro / Max（具体价格见套餐页）
- 限售中（每日 10:00 释放库存）

### 4. 龙虾套餐（团队协作版）
- 体验卡：39 元/月，含 3500 万 tokens GLM-5-Turbo
- 进阶卡：99 元/月，含 1 亿 tokens GLM-5-Turbo
- 团队版：联系销售，按需定制
- 当前已售罄，暂无再次开放计划

### 5. 企业级服务
- 私有化部署：GLM-4.6 全方案 110 万元/年（5840 算力单元 + 10 亿训练 tokens）
- GLM-4.5-Air 方案：50 万元/年
- 增购算力：60-70 元/算力单元/天
- 模型微调服务：10000 元/人/天
- 专属知识库、定制化方案：联系销售

### 6. 用户权益体系（V0-V3）
- V0（0-2000 积分）：基础服务
- V1（2000-10000 积分）：并发权益
- V2（10000-50000 积分）：更高并发
- V3（50000+ 积分）：最高并发
- 积分获取：消耗现金余额 1:1 兑换积分
