## 特殊能力

### GLM-5.1 独有能力
- 可独立工作 8 小时的 Agentic 模型
- 200K 上下文 + 128K 输出
- Coding 对齐 Claude Opus 4.6
- 支持深度思考 + Agent 模式

### GLM-5 独有能力
- 编程对齐 Claude Opus 4.5
- 200K 上下文 + 128K 输出
- 沉思模式（深度推理）

### GLM-4.7 独有优势
- LM Arena 开源第一、全球第四
- OpenRouter 趋势榜第一
- 性价比极高

### GLM-4-Long 独有能力
- 100 万 token 超长上下文
- 适合处理超长文档、长代码库

### GLM-5V-Turbo 独有能力
- 最强视觉 Coding 能力
- 200K 上下文 + 128K 输出
